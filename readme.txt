== Description ==

An extension for Visual Composer that restrict the content or block based on user role and display message for restricted role as well.

WP Bakery plugin must be installed and activated to have this element in editor.

Some of basic feature using this plugin:

*   Content restriction based on user role.
*   Dynamic heading tag for showing title of section and it will be hide/show based on selection.
*   Banner image upload for restricted section.
*   Set the custom description for section.
*   Display message for logged in user or non-logged in user.
*   You can show custom message for different users if users is not allowed.

== Installation ==
<strong>To install "Restrict Content for WP Bakery" using the built-in plugin installer: </strong>
1. Go to Plugins > Add New.    
2. Type in "Restrict Content for WP Bakery" in Search Plugins box or click a tag link below the screen.    
3. Find the "Restrict Content for WP Bakery" in the list.    
4. Click Install Now to install the WordPress Plugin.
5. Click Activate to activate it.
6. Go to 'Content Editor' in admin and click on 'Add Element' in WP Backery block.   
7. Select 'Restricted Block' section, select it and add your content.
8. Set your own settings and add block.

<strong>To install "Restrict Content for WP Bakery" manually:  </strong>
1. Download your WordPress Plugin to your desktop.   
2. If downloaded as a zip archive, extract the Plugin folder to your desktop.   
3. With your FTP program, upload the Plugin folder to the wp-content/plugins folder in your WordPress directory online.   
4. Go to Plugins screen and find the "Restrict Content for WP Bakery" in the list.   
5. Click Activate to activate it.   
6. Go to 'Content Editor' in admin and click on 'Add Element' in WP Backery block.   
7. Select 'Restricted Block' section, select it and add your content.
8. Set your own settings and add block.

== Frequently Asked Questions ==

= Does this plugin require any other plugin? =

Yes, WPBakery Page Builder plugin should be activated for this plugin.

= Can i hide block based on User Role? =

Yes, You can select the role to show the block at block editor dropdown 'Available for this role' dropdown.

= Can i show custom message if content is not visible for other roles? =

Yes, You can select 'Show Restricted Message' dropdown 'Yes'.

= Can i set custom message if content is not visible for other roles? =

Yes, You can set custom message at 'Restricted Message'.
